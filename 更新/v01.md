## 本次任务：v1 功能升级

### 1. 复习算法升级（最高优先级）
把当前的 5 级时间间隔替换为标准艾宾浩斯曲线：
- 间隔：1, 2, 4, 7, 15, 30 天
- 每个单词记录 `level`（0–6）和 `nextReview`（timestamp）
- 答对 → level+1，更新 nextReview；答错 → level 重置为 0
- 不要破坏现有 localStorage 数据结构，做好兼容迁移

### 2. 统计栏扩展（2行×4列）
第一行：今日已学 / 已掌握 / 待复习 / 学习时长
第二行：连续打卡 / 总词库 / 正确率 / 剩余新词
数据实时更新，不需要刷页面。

### 3. 多学习模式
在学习界面顶部加模式切换 tab，三种模式共用同一套单词队列：
- 闪卡模式：显示单词 → 点击翻转 → 自评掌握/模糊/不会
- 选择题模式：4 个选项，1 个正确释义 + 3 个干扰项（从词库随机抽）
- 拼写模式：显示释义 → 用户输入单词 → 回车校验，忽略大小写

### 4. 学习日历
底部加一个日历图标按钮，点击弹出面板：
- 展示近 90 天的热力图（类 GitHub contribution graph）
- 颜色深浅对应当天学习单词数（0/1-5/6-15/16+，四档）
- 点击某天格子，展示当天：学了几词、对了几题、用时多久

**dailyStats 存储格式必须统一为：**
```json
{
  "2026-05-27": { "learned": 12, "reviewed": 30, "correct": 25, "duration": 1800 }
}
```
key 用 YYYY-MM-DD，方便后续迁移到后端数据库。
