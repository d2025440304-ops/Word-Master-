在左栏统计数据和工具栏之间，新增一个"单词本"折叠区。
不新增页面，不改任何 JS 算法和数据结构。

## 单词本结构

折叠区标题行：
- 左侧文字"单词本"
- 右侧一个展开/收起箭头，默认收起
- 展开后最大高度 240px，超出滚动，不撑开左栏

顶部 tab 切换（两个，小尺寸）：
- "全部"：显示所有 level > 0 的词
- "已掌握"：只显示 level >= 6 的词

每行格式：
- 左侧：单词 + 空格 + 释义，单行截断超出部分
- 右侧：一个小圆点，level >= 6 显示绿色（var(--success)），否则显示 var(--text-tertiary) 浅灰色
- 字号 13px，行高 32px，hover 背景轻微高亮

## 数据来源

从 appData.words 遍历，level > 0 的词才显示。
需要从当前激活词库（getActiveWords()）找到对应的词义，用 word.m 字段。
按 level 从高到低排序，同 level 按字母排序。

## 约束
- 不改任何 JS 逻辑，只新增 DOM 和相关渲染函数
- 单词本数据在 updateStatsUI() 末尾调用一次 renderWordBook() 刷新
- 折叠状态存在内存变量里，不存 localStorage
